Projects

Projects are a key tool for learning new techniques, programming languages, and getting real work experience bringing ideas into a working environment. Having non work or academic projects provides opportunities to investigate new ideas and gives an outlet for creativity.

GrubNums

Purpose:
  • The premise of GrubNums is users sharing images of food they've eaten. Other users rate the food based on specific criteria, guess the calories/macronutrients, and leave comments. Users would visit GrubNums not to find out what restaurant to eat at (although they could) but instead to find out what is the best rated food item at each restaurant and an estimate of the foods particular calories.
Results:
  • Built a login and register system and CMS (Content Management System) from scratch.
  • Designed the database architecture and developed data flow diagrams, entity relationship diagrams and data structure diagrams.
Skill Sets:
  • Learned PHP, Apache, JQuery, JavaScript, and AJAX. Enhanced my expertise in and knowledge of HTML, CSS, MySQL. Improved skills in Relational Database Management.
  • Project Management.
  • Persistance in the face of set backs.

MenuFind - Android

Purpose:
  • Allow users to find foods meeting their macronutrient goals:
    • Food Search: Search the database using parameters from Calories, Carbs, Fat, Protein, Fiber, Food Name, and Restaurant.
    • Enhanced Search: Allow users to return results ordered by various parameters.
    • Query Results: Allows users to see a list of results fitting their query.
    • Results Drill Down: Provided detailed information of the food and allows users to edit the result if necessary.
    • Mapping: Allows users to launch maps for directions to the restaurant selected in the search
    • Add Food: Allows users to add new foods to the database which will be returned in the search.
Results:
  • As of Oct 2014 IIFYM MenuFind has over 1000 downloads and a 4+ star ratings.
  • Received positive feedback from users as well as requests for an iOS version.
Skill Sets:
  • Self taught Java and Android Development using Eclipse. Implemented multi level SQL queries within my application.
  • Manipulating, cleansing & processing data in Java and SQL.
  • Google Play publishing.

MyFitnessPal Analyzer - Shiny by RStudio

Purpose:
  • Improve the reporting functions provided by MyFitnessPal. Key functions include:
    • Food Recommendations: a Monte Carlo algorithm which analyzes your recent diet history and recommends foods to hit your goals.
    • Macro Analysis: Summaries for the Carbs, Protein, Fat, and Fiber eaten.
    • Food Analysis: time series analysis of each food eaten during a specified time period
    • Food Data: aggregation of foods eaten
  • Deploy Shiny Server on Amazons EC2.
Results:
  • Built a scraper which allows improved analysis of MyFitnessPal data.
  • Allows any user to input their username and the data range which they want to analyze.
Skill Sets:
  • Improved R programming skills including: Regular Expressions, Web Scraping, Data Visualization, and Dashboard Development.
  • Learned AWS and EC2. Improved UNIX skills.
